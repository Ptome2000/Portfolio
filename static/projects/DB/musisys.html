Database Musisys Project

Description

This project is an advanced database designed to facilitate the robust management of music festivals. It handles a wide range of data related to festival editions, stages, participants, technicians, spectators, and ticketing. Divided in 2 parts where the first was the design and development of the database, while the second was the optimization of a similiar solution

The project aims to streamline festival organization, enhance operational efficiency, and ensure a comprehensive record-keeping process for all festival-related activities.

Key features include:
Festival Edition Management:
  • Track festival editions with unique identifiers, names, locations, venues, dates, and capacity
Stage Configuration:
  • Manage up to seven stages per festival, each with a unique code and name
Technician Assignment:
  • Register and track stage setup technicians and participant-associated roadies
Participant Scheduling:
  • Record participant details, performance times, styles, fees, and song sequences
  • Include guest appearances by other participants
Spectator Categorization:
  • Identify spectators as journalists, guests, or paying spectators, recording specific attributes for each category
Ticketing System:
  • Manage ticket sales and distributions, including single-day and multi-day tickets
  • Allow ticket returns with transaction logging
Operations Log:
  • Maintain a history of all operations and transactions within the system
Interview Scheduling:
  • Schedule and record interviews between journalists and participants
Optimizations:
  • Improved the relational database model for enhanced performance and efficiency
Automations:
  • Automated processes for routine tasks to streamline operations and reduce manual effort
Search Capabilities:
  • Set of advanced search functionalities to facilitate quick and accurate retrieval of information
Web-based Prototype:
  • Demonstration prototype using HTML and PHP to showcase the system's features and functionalities in a web-based interface
